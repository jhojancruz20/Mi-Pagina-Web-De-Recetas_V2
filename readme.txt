# Proyecto Web Receta de Cocina V1

Este proyecto es una aplicación web para gestionar y compartir recetas de cocina. 
Permite a los usuarios buscar, agregar, editar y eliminar recetas, 
así como ver detalles e imágenes de cada platillo.



## Características

- Listado de recetas con búsqueda y filtrado.
- Visualización de detalles de cada receta.
- Agregar, editar y eliminar recetas.
- Subida de imágenes para cada receta.
- Interfaz intuitiva y responsiva.
